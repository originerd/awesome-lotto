# Awesome Lotto
A collection of awesome lotto lottery number generators ~~Just for fun~~

## Format
- Example - `3, 7, 26, 31, 32, 41`
  - Pick 6 numbers from 1 to 45 without repetition
  - Insert ', ' between sorted numbers
